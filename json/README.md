# JSON

Há um arquivo JSON para cada versão descrita acima. Os arquivos JSON estão codificados em UTF-8 e possuem a seguinte estrutura:

## Curiosidade

```bash
{
  "1-1": {
    "curiosidade": [
      {
        "texto": "..."
      },
      {
        "texto1": "..."
      }
    ]
  },
  "1-2": {
    "curiosidade": [
      {
        "texto": "..."
      },
      {
        "texto1": "..."
      }
    ]
  }
},
...
}
```

## Eventos

```bash
 {
  "1-1": [],
  "1-2": [],
  ...
  "12-31": []
 }
```
